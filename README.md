# Hospital Management System (HMS)

A Java-based Hospital Management System using JDBC and MySQL.

## Features
- Patient Management
- Doctor Management
- Appointments
- Billing System with JDBC Transactions

## Technologies Used
- Java (JDK 17)
- MySQL 8
- JDBC
- Java Swing

## Key Highlight
Billing module uses JDBC transactions (commit & rollback) to ensure data consistency.

## How to Run
1. Create database `hospital_db`
2. Run SQL files
3. Update DB credentials in DBConnection.java
4. Run Main.java
